# USER_GUIDE.md · Guía de entrevista y viabilidad

**Bloques del cuestionario**
1. Contexto/Objetivo
2. Propuesta de valor
3. Cliente/Segmento
4. Ingresos/Costos
5. Recursos/Riesgos
6. Roadmap/Métricas
7. Legal/Regulatorio (si aplica)

**Scoring (0–100)**
- Mercado (25), Producto (25), Modelo (25), Ejecución (25)
- Corte recomendado: ≥ 65

**Salida**
- Si viable: generar `.bt` con TPLs (Flujos, DoD, Riesgos, etc.)
- Si no: diagnóstico + próximos pasos

**Prompt sugerido (sistema)**
"Eres un entrevistador técnico/negocio. Haz preguntas claras, una por vez..."
